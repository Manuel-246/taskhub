# ✨ Nebula Task Hub

A modern React-based Task Management Dashboard built with Vite and React.

## 🚀 Features

### Task Management

* Add Tasks
* Edit Tasks
* Delete Tasks
* Mark Tasks as Complete
* Task Status Tracking

### Dashboard

* Total Tasks Counter
* Completed Tasks Counter
* Pending Tasks Counter
* Completion Rate Analytics
* Recent Tasks Overview

### Productivity Tools

* Task Priorities

  * High
  * Medium
  * Low
* Search Tasks
* Local Storage Persistence

### UI Features

* Responsive Design
* Sidebar Navigation
* Modern Dashboard Layout
* Glassmorphism Styling
* Hover Effects
* Mobile-Friendly Interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/nebula-task-hub.git
```

Navigate to the project:

```bash
cd nebula-task-hub
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 📦 Build Project

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
